Good Look Browbar Website

Deployable static website for GitHub Pages or any standard hosting.

Included pages:
- index.html, main salon website
- products.html, ecommerce style product page
- admin.html, demo admin panel

Features:
- Modern responsive salon design
- Product page with add to cart
- Coupon code GL10
- Checkout with terms agreement
- Bank transfer payment instructions
- Product order storage in browser localStorage
- Admin panel for appointments and orders
- Dummy products and placeholder product images

Important:
This is a static front-end version. Orders and appointments are saved in the browser only. For real business use, connect the website to a database, secure admin login, email notifications, and a payment gateway.
