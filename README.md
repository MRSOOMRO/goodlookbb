Good Look Browbar Website

Files included:
- index.html
- style.css
- script.js
- assets/logo.png

How to deploy on GitHub Pages:
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Open Settings.
4. Open Pages.
5. Select Deploy from branch.
6. Select main branch and root folder.
7. Save.

How to edit:
- Update phone, WhatsApp, email and address inside index.html.
- Update prices inside the pricing section.
- Replace assets/logo.png with your final logo if needed.
